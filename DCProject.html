About the Project

The Daily Cart is a sleek e-commerce platform designed for online grocery shopping. It offers a seamless shopping experience with a dynamic cart system, secure checkout, and an intuitive user interface.

Technologies Used

  • HTML, CSS, JavaScript
  • Node.js (Backend)
  • MySQL (Database)
  • JWT Authentication

Product Categories

Browse through different categories like fruits, vegetables, dairy, and more.
